# Auto Buy (Card2K Discord Bot)

## Giới thiệu
Đây là dự án Discord Bot tích hợp Card2K để nạp thẻ cào tự động.

## Công nghệ sử dụng
- Node.js
- discord.js v14
- axios
- dotenv
- express
- groq-sdk

## Cài đặt
```bash
npm install
```

## Chạy bot
```bash
npm start
```

## Chạy ở chế độ phát triển
```bash
npm run dev
```

## Cấu trúc dự án
- `index.js` - File khởi động chính
- `.env` - Biến môi trường
- `package.json` - Cấu hình dự án

## Package
Tên package: `card2k-bot`
Phiên bản: `1.0.0`
