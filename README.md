# Auto Buy (Stela Auto Buy Bot ) 


## Giới thiệu
Đây là dự án Discord Bot tích hợp để nạp thẻ cào .

## Công nghệ sử dụng
- Node.js
- discord.js v14
- axios
- dotenv
- express
- groq-sdk

## Cài đặt
```bash
npm install
```

## Chạy bot
```bash
npm start
```

## Chạy ở chế độ phát triển
```bash
npm run dev
```

## Cấu trúc dự án
- `index.js` - File khởi động chính
- `.env` - Biến môi trường
- `package.json` - Cấu hình dự án

Tên package: `stela-auto`
